# Bilal Gadget Store Website 🛍️

A modern multi-page static website for a gadget store, featuring product listings, store information, and a contact form.

## 🚀 Features
- Multi-page layout (Home, Products, About, Contact)
- Product cards with images and pricing
- Responsive navigation with hamburger menu
- About section with store details & stats
- Contact form for user inquiries
- Clean and responsive UI design

## 🛠️ Built With
- HTML
- CSS
- JavaScript
- Font Awesome

## 📂 Project Structure
- index.html → Home page
- products.html → Products listing
- about.html → About store
- contact.html → Contact form
- style.css → Styling
- script.js → Navigation logic

## 📚 What I Learned
- Building multi-page websites
- Responsive layouts using CSS Grid
- Navigation handling with JavaScript
- UI design with images, icons & fonts
- Structuring real-world static websites

## ▶️ How to Run
Open `index.html` in your browser.
